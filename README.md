东京IT技术交流
----

线下交流技术的Blog，希望能获得更多朋友的关注和参加。

# 安装
- `git clone <repo>`
- `cd <dir>`
- `npm install -g hexo`
- `npm install`
- `git clone https://github.com/hexojs/hexo-theme-light.git themes/light`

# 写文章
- `hexo new <title>`

# 本地确认
- `hexo generate`
- `hexo server`
- 打开 `0.0.0.0:4000`

# 部署
- `hexo deploy -g`
